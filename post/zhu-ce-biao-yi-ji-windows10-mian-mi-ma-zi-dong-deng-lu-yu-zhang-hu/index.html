注册表一击，Windows10 免密码自动登录域帐户

2019-07-24 1 min read 
Windows Registry Editor Version 5.00
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon] 
"AutoAdminLogon"="1"
"DefaultUserName"="域帐户"
"DefaultDomainName"="域"
"DefaultPassword"="域帐户密码"
