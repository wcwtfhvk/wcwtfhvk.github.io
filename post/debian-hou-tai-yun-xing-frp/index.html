Debian 后台运行FRP

2019-05-05 技术经验

用 supervisor
先安装 supervisor
sudo apt install supervisor
创建 supervisor frps 配置文件
在 /etc/supervisor/conf.d
创建撇脂文件 frp.conf

[program:frp]
command = /root/frp/frps -c /root/frp/frps.ini
autostart = true

重启supervisor服务
systemctl restart supervisor
查看supervisor运行状态
supervisorctl status
