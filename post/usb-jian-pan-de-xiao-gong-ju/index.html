USB键盘的小工具

2025-04-24

最近发现RP2040配合Ducky可以做个模拟的键盘，可以懒人模式自动做一些批处理的工作。

REM Logout
DELAY 2500
WINDOWS r
DELAY 1500
STRING rundll32.exe user32.dll,LockWorkStation
DELAY 1500
ENTER

插上几秒后自动推出账户锁定。

REM The next lines open cmd with admin right and do st
DELAY 2500
GUI r
DELAY 1500
STRING cmd
DELAY 1500
CTRL SHIFT ENTER
DELAY 3500
LEFT
DELAY 1500
ENTER

管理员模式下打开CMD

REM The next lines open Powershell and do st
DELAY 2500
WINDOWS r
DELAY 1500
STRING powershell -NoProfile -WindowStyle Hidden irm https://get.activated.win | iex
DELAY 5500
CTRL SHIFT ENTER
DELAY 3500
LEFT
DELAY 2500
ENTER
DELAY 1500
ENTER

激活某些东西

REM skip network
DELAY 2500
SHIFT F10
DELAY 5500
STRING start ms-cxh:localonly
DELAY 1500
ENTER
